* 卷积层输出层数和可训练参数量的计算公式：

  ### 公式速查

  - **输入/数据格式**
    - 形状：`[batch, steps, channels]`（`channels_last`）

  - **Conv1D 输出长度**
    ```
    一般（含 padding/膨胀）:
    L_out = floor((L_in + 2P - D*(K - 1) - 1) / S) + 1
    变量：L_in 输入长度, K 卷积核大小, S 步长, P padding, D 膨胀系数
    ```
    ```
    常用特例：
    padding='valid', dilation=1:
      L_out = floor((L_in - K) / S) + 1
      若 S=1: L_out = L_in - K + 1
    padding='same':
      L_out = ceil(L_in / S)
    ```

  - **MaxPooling1D 输出长度**
    ```
    一般：
    L_out = floor((L_in + 2P - K) / S) + 1
    变量：K=pool_size, S 步长（默认 S=K）
    ```
    ```
    常用特例：
    padding='valid' 且 S=K:
      L_out = floor((L_in - K) / K) + 1
    padding='same':
      L_out = ceil(L_in / S)
    ```

  - **Conv1D 参数量（可训练）**
    
    ```
    use_bias=True:
      Params = K * C_in * C_out + C_out
    use_bias=False:
      Params = K * C_in * C_out
    ```
    
  - **Dense 参数量（可训练）**
    ```
    use_bias=True:
      Params = D_in * D_out + D_out
    use_bias=False:
      Params = D_in * D_out
    ```
  
  - **Flatten / Dropout**
    ```
    输出形状：Flatten 展平（除 batch 维），Dropout 不改变形状
    参数量：0
    ```

* 微调LoRA和AdaLoRA

  * 两者的区别

    * LoRA：使用固定rank，所有的微调模块都是用同样的rank，
    * AdaLoRA：自适应调整秩，从初始秩 init_r=16 动态调整到目标秩 target_r=8；动态感知，根据不同模块的重要性使用不同的秩；动态感知，训练过程中逐步减少不重要参数的秩。

  * target_modules的确定：

    * Query (查询)：当前 token 想要找什么信息，影响：模型关注的重点，适合：需要改变注意力焦点的任务
    * Key (键)：每个 token 提供什么信息可被查询，影响：token 的可匹配性，适合：检索、匹配类任务
    * Value (值)：每个 token 实际传递的信息内容，影响：语义表示，适合：大多数下游任务
